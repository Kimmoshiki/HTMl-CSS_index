블럭방식의 요소 - 제목태그, 문단태그, 수평선, 목록, div, 시멘틱구조태그.....

p태그인 나는 블럭요소입니다. 왼쪽에서 오른쪽까지 내용을 무조건 자치하지요. 블럭요소는 줄바꿈을 하지않아도 자동으로 줄이 변경됩니다. 또한 가로크기, 세로크기값을 설정하여 크기를 지정할 수 있습니다. 블럭요소의 종류로는 p, h1~h6, div, header, nav, main, section, article, aside, footer, address, dl, dt, dd, ul, li, ol, form, fieldset, hr, table가 있습니다.

위 문단요소의 성격이 변경되어 현재 글자가 올라갑니다.

인라인요소

span태그인 나는 인라인 요소입니다. 글자가 있는곳까지만 영역을 차지하지요. 블럭요소와는 다르게 자동으로 줄이 변경이되지 않고 br요소를 사용하여 줄바꿈을 합니다. 인라인요소는 크기값을 가질 수 없습니다. 대신 display속성을 block또는 line-block으로 변경하면 가능합니다. 대표적인 태그로는 img, b, strong, sup, sub, input, label, button, textarea....등이 있습니다. span태그는 태그자체의 의미는 없지만 글자를 묶어서 서식을 적용하고자 할 때 사용. 태그자체의 의미는 없지만 글자를 묶어서 서식을 적용하고자 할 때 사용. 하이퍼 링크 하이퍼 링크 글자에 밑줄긋기 글자강조하기 취소선

박스모델(box model)이란? 블럭요소와 인라인요소가 가질 수 있는 크기와 마진, 패딩, 테두리값을 사용하여 배치하는 개념을 말한다.

사용속성 - 크기속성,테두리,마진,패딩
width, height(p.162) : 크기를 지정할 때 사용하며,px,%단위를 사용한다.
border(p.167) : 테두리를 지정하고자 할 때 사용하며 축약법으로는 'border:선두께 선모양 선색상'식으로 작성한다.
margin(p.164) : 선택한 요소를 기준으로 바깥쪽으로 여백을 설정할 때 사용.
padding(p.165) : 선택한 요소를 기준으로 안쪽으로 여백을 설정할 때 사용.

        세로전체크기=패딩+마진+테두리+세로크기
