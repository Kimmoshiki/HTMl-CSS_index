02_요소의 배치방법 - position(포지션)p.188

포지션(position) 속성이란? 레이아웃 작업시 사용자가 부모요소 박스안에서 원하는 콘텐츠를 선택하여 배치하기 위한 속성으로서 static(요소가 가지고 있는 기본 위치값),abrolute(절대배치), relatuve(상대배치),fixed(고정배치), inherit(상속)의 종류가 있다

포지션 속성 사용시 위치를 지정하기 위해서는 top, bottom, right, left속성을 사용한다.

포지션 속성을 사용하면 float속성처럼 콘텐츠와 콘텐츠끼리 겹침현상이 나타나는데, 위, 아래 순서을 변경하고자 할 떄는 z-lndex속성을 사용한다.

z-index속성을 먼저 작성된 태그가 z-index값이 0이며, 나중에 작성된 태그 순서대로 번호가 자동으로 매겨진다. z-index속성을 사용하여 위/아래 순서도 변경이 가능하다.

1. 절대배치(adsolute)

2. 상대배치(relative)

3. 고정배치(fixed)
